# NEOMIND – Sovereign Intelligence Grid  
**المنظومة السيادية لاستخبارات المخاطر والتوأم الرقمي الوطني**

> نسخة تجريبية للعرض أمام لجنة **هاكاثون أبشر طويق 2025**  
> هذه النسخة لا تعتمد على بيانات حقيقية، بل على بيانات محاكاة يتم توليدها داخليًا لأغراض العرض فقط.

---

## 1. فكرة النموذج (High-Level Overview)

NEOMIND هي منصة سيادية لاستخبارات المخاطر والتوأم الرقمي الوطني، تقوم بـ:

- جمع البيانات من مصادر ميدانية متعددة (دوريات، فرق ميدانية، كاميرات، حساسات، بلاغات…).
- تحليل الأنماط وحالة المخاطر عبر **محرك الأنماط PSIC**.
- تغذية **المحرك السيادي المركزي CORE** برؤوس المؤشرات لتحديث صورة القرار.
- دعم لوحات القيادة العليا بصورة حية على خريطة وطنية / مدينية (في النموذج الحالي: مدينة الرياض).
- إتاحة طبقة نماذج تعلم آلي **(ML Engine – Skeleton)** للتوسع المستقبلي.

هذه النسخة تركّز على **تجربة الاستخدام أمام اللجنة** أكثر من اكتمال جميع الوظائف الخلفية.
